* ===== README =====
Project: Simple To-Do App

Description:
A basic To-Do list application using Java Swing. Users can add and delete tasks.

Features:
- Add task
- Delete task
- Simple GUI

Technologies:
- Java
- Swing (JFrame, JButton, JTextField, JList)

How to Run:
1. Compile:
   javac SimpleTodoApp.java
2. Run:
   java SimpleTodoApp

Concepts Used:
- Swing components
- Event Handling
- Layout Manager (BorderLayout)

==================== */
